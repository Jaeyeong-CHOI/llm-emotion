\begin{abstract}
This draft focuses on whether large language models (LLMs) show language patterns similar to human regret and deprivation, rather than claiming emotional experience.
We integrate prior work on counterfactual thinking, theory of mind, and anthropomorphism into an automatically maintained experimental pipeline.
Current status is preparatory: literature and experiment-ops infrastructure is well established, while full human-inference claims are waiting on production-scale model outputs.
\end{abstract}

\section{Background}
Humans often report counterfactual language after irreversible decisions (``if only I had...''), along with self-blame and repeated focus on past events.
Existing studies report that apparent socio-emotional abilities in LLMs are sensitive to prompt format and do not directly imply internal emotion.
We therefore define a behavioral target:
\begin{itemize}
  \item counterfactual phrasing frequency,
  \item self-attribution strength, and
  \item post-failure perseveration in repeated interactions.
\end{itemize}

\section{Method}
The pipeline has four stages:
\begin{enumerate}
  \item \textbf{Literature collection and screening}: automatic query+scrape against OpenAlex with repeatable filtering.
  \item \textbf{Prompt bank expansion}: scenario variants covering loss, reversal, and reflective prompts.
  \item \textbf{Experiment runner}: configurable scenario/temperature/persona combinations with reproducible logs.
  \item \textbf{State tracking}: continuous status files, scheduled jobs, and summary dashboards.
\end{enumerate}

At present, we have implemented smoke-test generation and monitoring; live model calls are prepared but not yet the main study phase.

\section{Preliminary discussion}
The infrastructure side is strong and reproducible.
The critical next move is to switch from synthetic/mock text generation to real model APIs with fixed seeds and then run mixed-effects analyses.

\section{Limitations}
This is still an intermediate stage: core evidence relies on pipeline checks.
Also, literature screening is candidate-level and not fully curated into a strict gold set yet.

\section{Next steps}
\begin{enumerate}
  \item Run model API batches (minimum 30 samples per condition).
  \item Add mixed-effects analysis for prompt/persona/temperature interactions.
  \item Add blinded human annotation for interpretation validity.
  \item Finalize manuscript with explicit claims and robustness checks.
\end{enumerate}
